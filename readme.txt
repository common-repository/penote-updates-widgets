== Description ==

Allows for multiple penote widgets to be displayed.
<a href="http://penote.com/signup/?ref=wordpress">Penote.com</a> is a social web service that allows the subscriber to broadcast short messages or share notes to other subscribers of the service.

== Installation ==
Support Wordpress, WordpressMU and Wordpress 3.0

=Standard MU Plugins installation.=

1. Upload 'penote_widgets.php' to the '/wp-content/mu-plugins/' directory

2. It's done!

=Standard Plugins installation.=

1. Upload 'penote_widgets.php' to the /wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Enabled the widgets and setting at Appearance->Widgets

3. No more! Just look your site and admin panel. :)
